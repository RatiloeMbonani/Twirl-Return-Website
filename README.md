# Twirl & Return – Matric Dress Rental Website

## 📖 Overview
**Twirl & Return** is a responsive web platform designed to showcase and rent elegant matric dance gowns. The website combines luxury branding with accessibility, providing users with an engaging browsing experience while exploring a curated collection of gowns available for rental.
The platform includes a gown gallery, enquiry form, FAQ section, and an interactive light/dark theme toggle. The project focuses on delivering a visually appealing and user-friendly interface while demonstrating strong front-end development skills.This project was built using html,css and javascript, with an emphasis on responsive design, usability, and interactive functionality.

## ✨ Features
Responsive Navigation
* Desktop horizontal navigation bar with hover effects
* Mobile-friendly hamburger menu
* Dropdown navigation links for smaller screens
  
Theme Toggle (Light / Dark Mode)
* Users can switch between light and dark themes
* Theme preference is saved using local storage
* Accessible and keyboard-friendly toggle button
  
Hero Section
* Full-width background video
* Brightness filter applied for improved readability
* Creates an immersive visual experience

Dress Gallery
* Responsive grid displaying available gowns
* Each card includes:
  * Image
  * Dress name
  * Description
  * Size
  * Price
  * Availability
* Hover zoom effect for enhanced user engagement

About & Benefits Section
* Clear brand story and mission
* Informative sections explaining the benefits of renting gowns
* Clean layout with elegant typography

Enquiry Form
* Allows users to send rental enquiries
* Client-side form validation using JavaScript
* Displays confirmation alert upon successful submission
* Automatically resets the form

FAQ Section
* Common questions answered clearly
* Collapsible question and answer structure
* Improves user trust and transparency

Footer
* Contact information
* Social media links
* Responsive layout for different screen sizes

## Technologies Used
* **HTML5** – Structure and content
* **CSS3** – Styling, layout, and responsive design
* **JavaScript** – Interactive functionality and form validation

## Project Structure
index.html – Main homepage containing layout and content
style.css – All styling including layout, typography, and themes
myscript.js – JavaScript for form validation, theme switching, and mobile navigation
README.md – Project documentation


## JavaScript Functionality
Form Validation
* Ensures name, email, and message fields are not empty
* Displays a thank-you alert
* Resets the form after successful submission

Theme Toggle
* Retrieves the saved theme from local storage
* Toggles between light and dark modes
* Updates the `data-theme` attribute on the root element

Mobile Navigation Toggle
* Shows or hides navigation links on small screens
* Adds or removes the show class from the navigation menu

FAQ Interactivity
* Converts FAQ paragraphs into clickable questions
* Allows answers to expand or collapse when clicked


## Responsive Design
The website adapts to different screen sizes using CSS media queries.

Mobile Adjustments (below 768px)
Navigation
* Navigation menu stacks vertically
* Hamburger menu appears
* Links display as dropdown items

Hero Section
* Reduced height
* Simplified layout

Gallery
* Grid switches to a single column
* Dress cards expand to full width

Forms and FAQ
* Stacked layout
* Adjusted spacing and font sizes

Footer
* Sections stack vertically
* Centered text for readability

## How to Run the Project

1. Download or clone the repository(git clone https://github.com/RatiloeMbonani/twirl-return.git)
2. Open the project folder.
3. Ensure the following files are present:
* index.html
* style.css
* myscript.js
4. Double-click index.html or open it in your browser.
5. Explore the website features such as the gallery, theme toggle, and enquiry form.
